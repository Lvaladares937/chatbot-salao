# 🤖 Bot IA para Salão Vailson's Hair & Makeup

Bot inteligente para atendimento humanizado no WhatsApp usando Google Gemini.

## ✨ Funcionalidades

- 🗣️ Conversa natural como um atendente real
- 📅 Agendamento inteligente de horários
- 💇 Catálogo completo de serviços
- 👥 Informações sobre profissionais
- 📍 Localização e contatos
- 🔄 Cancelamento e remarcação
- 💾 Memória de preferências dos clientes
- 📱 Integração com Google Calendar (opcional)

## 🚀 Instalação

```bash
# Clone o repositório
git clone [seu-repositorio]
cd salao-vailson-bot-ia

# Instale as dependências
npm install

# Configure as variáveis de ambiente
cp .env.example .env
# Edite o .env com sua chave do Gemini

# Inicie o bot
npm start
